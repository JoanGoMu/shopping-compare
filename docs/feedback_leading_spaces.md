Never start paragraphs, string fields, or any piece of text content with a leading space (e.g. `" Some text"`).

**Why:** Leading spaces before text are a classic AI writing tell. Joan flagged this as something to avoid. It makes content look machine-generated.

**How to apply:** When generating any content - tool descriptions, blog post paragraphs, comparison verdicts, taglines, features, pros, cons - always trim leading spaces. Applies to both plain text fields and HTML content (e.g. `<p> text` should be `<p>text`).
